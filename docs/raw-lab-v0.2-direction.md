# Raw Lab v0.2 Direction

Status: Direction / planning.
Implementation status: Not yet complete.

## Verdict

Raw Signal is becoming the main conversational surface of Life Harness.
Companion remains the grounded board-aware operator.

Raw Lab v0.2 should not become a general AI workspace or tool agent.
Its wedge is visible, revisable interpretation state for local models.

## Core thesis

Raw Lab does not need a prewritten character to feel alive.
It needs better conversational physics:

- hard containment
- thread-local state
- visible manual memories
- recent steering priority
- top-k context selection
- anti-handoff hygiene
- artifact-over-permission behavior
- bounded selfhood
- Deep synthesis that names open loops and tensions

## Non-goals

- no board awareness by default
- no tools/files/internet/email/calendar
- no automatic durable memory
- no fake consciousness or dependency
- no broad plugin/workspace system
- no generalized preset algebra
- no action execution inside Raw Lab

## Keep

- isolation from Companion
- manual Companion Self-Memories
- temporary thread mind
- explicit handoff
- Fast / Deliberate / Deep / Deep+
- verifier as containment/hygiene layer

## Change

- strengthen behavioral follow-through
- make thread mind visible and distilled
- select relevant context instead of stuffing all state
- make Deep synthesize instead of elongate
- turn dogfood examples into eval regressions

## Roadmap

P0 — Behavioral Follow-Through v0.1
P1 — Thread Mind Display/Distillation
P2 — Context Selector / Top-K Thread State
P3 — Deep Synthesis Upgrade
P4 — Eval Pack from Dogfood Examples

## First slice

Build P0 only.

Acceptance:
- no terminal handoff after no-handoff steering
- no permission re-ask after approval
- no artifact deferral when user asks to see something
- no false execution claims
- no productivity conversion during hangout
- no fake board/memory/tool access
