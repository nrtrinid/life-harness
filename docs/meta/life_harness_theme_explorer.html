Theme 01
Signal / Static
Hot cards transmit clearly. Cold cards go to static. The board reads your life state like a radar station — no judgment, just signal.
—Inter + JetBrains Mono
—Electric blue as the only accent
—Signal-strength warmth system
—Radiohead / Kid A energy
—Cold = static · Hot = clear signal
Theme 02
Field Ops
Cards as dossiers. Board as command room. Every active project is an open operation. Every parked idea is properly filed.
—Barlow Condensed + Inter
—Brass / dark olive palette
—Stamp-style card status labels
—Square corners throughout
—Physical mission board feel
Theme 03
Thermal Hud
Warmth is literal: card accents shift by temperature. Cold = blue. Warm = yellow. Hot = orange. The board is a heat map of your momentum.
—Rajdhani + Inter (technical)
—Color temperature maps to warmth
—Card borders shift blue → orange
—Esports HUD / targeting feel
—Most visceral of the four
Theme 04
Amber Terminal
Your personal OS. Log entries feel like shell output. Single amber spectrum — no other colors, warm black, completely focused.
—JetBrains Mono exclusively
—Single amber spectrum, zero other hues
—Logs feel like shell output
—Pounce styled as a command
—Fits your cybersecurity context
