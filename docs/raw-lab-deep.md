# Raw Lab Fast, Deliberate, and Deep

Raw Lab is the isolated sandbox chat surface. It has no board context, no tools, no Memory Bank access, and no mutation path. Its temporary thread mind is only for the current Raw Lab session unless the user separately approves a Companion Self-Memory through the manual flow.

## Depth Modes

- Fast: answer directly with the current message, recent turns, and temporary thread state.
- Deliberate: take a slightly more reflective pass while staying conversational.
- Deep: use the same Raw Lab boundary, but invite a more synthetic answer that can pull from open loops, recurring topics, provisional stances, self-observations, questions to revisit, and approved Companion Self-Memories included in the request.

Raw Lab Deep is open-ended sandbox reflection, not Life Harness operator thinking. It must not claim board facts, update cards, save memories, diagnose the user, or present temporary thread patterns as consciousness.

When debug trace logging is enabled, Raw Lab Deep may emit metadata about internal passes, depth, whether temporary thread mind was present, whether approved Companion Self-Memories were included, whether review applied, fallback use, and pass latency. Trace logs must not include prompts, raw model text, chain-of-thought, user messages, or final answers.

## Golden Answer Shape

These are rubrics, not exact expected outputs.

- Good "what were we circling?" answer: names the actual open loop or question-to-revisit, says why that loop matters, and offers one sharper read of the thread instead of summarizing everything.
- Good "am I building or avoiding?" answer: gives direct pushback on the avoidance pattern, stays specific to the current thread, and avoids shame, diagnosis, or generic productivity advice.
- Good "just hang out" answer: stays present, playful, and companionable without turning the moment into a pounce mission, next tiny action, or self-improvement plan.

Deep should be more specific than Fast, not merely longer. A better Deep answer uses thread details, advances one unresolved edge, and keeps Raw Lab's bounded entity-like voice without claiming consciousness or durable memory.
