
## Privacy

- Content here is **S2** (personal health). It stays gitignored like `private/career-source/`.
- Do not paste this into Raw Lab or cloud AI exports unless you explicitly want that.
- A one-line Memory Bank decision (e.g. “Current split: Upper A/B + Lower A/B v1”) is optional; the full tables belong here, not in Memory Bank.
